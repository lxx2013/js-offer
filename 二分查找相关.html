二分查找相关

1. 给定一个整数数组和目标值，找到和为目标值的两个元素。


输入[1, 0, -1, 9, 7, 5, -2, 4],6
输出[1,5]

思路: 控制左右各一个指针, 根据两个位置数字和和 target 的比较来移动指针, O(n)


        function search(a,target){
            a.sort((x,y)=>x-y);
            var l = 0, r = a.length-1;
            while(l < r){
                if(a[l] + a[r] < target) l++;
                else if(a[l] + a[r] == target) return [a[l],a[r]];
                else if(a[l] + a[r] >target) r--;
            }
            return []
        }
