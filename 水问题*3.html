1.简单版

leetcode

给定 n 个非负整数 a1，a2，...，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。 说明：你不能倾斜容器，且 n 的值至少为 2。 图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。 


    示例:
    输入: [1,8,6,2,5,4,8,3,7]
    输出: 49

算法思想: 从左右两个边界往中间夹的过程中取最大值,时间复杂度 O(n), 每次迭代时移动 min(left,right)这条边,因为面积=min(l,r)*(r-l), 移动短边的话才有可能让面积变大

2.复杂版

参考了https://blog.csdn.net/Lynn_Baby/article/details/80755070

题目描述：与上题相比,这一次 l 和 r 中间的东西算做了不能充水的障碍物. 


            示例:
            输入: [0,1,0,2,1,0,1,3,2,1,2,1]
            输出: 6

算法思想: 从左至右遍历,用 l[i] 来存放 i 左边最高的值; 从右向左遍历,用 r[i] 存放 i 右边最高的值, 这样 i 头顶的水面积就是(min(l[i],r[i])-input[i])*1

3.两桶水互相倒水

leetcode

题目描述：有两个容量分别为 x升 和 y升 的水壶以及无限多的水。请判断能否通过使用这两个水壶，从而可以得到恰好 z升 的水？ 如果可以，最后请用以上水壶中的一或两个来盛放取得的 z升 水。 


                示例:
                输入: x = 3, y = 5, z = 4
                输出: true
                输入: x = 2, y = 6, z = 5
                输出: False

算法思想: 题目等价于求 z = ax+by 其中 a,b 都是整数,此时只需求 x 和 y 的最大公约数的倍数+处理含0的情况即可,至于原因,参考欧里几德定理
